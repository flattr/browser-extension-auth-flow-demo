# Demo extension

## Usage

1. Install dependencies

    ```bash
    npm i -D
    ```
1. Run build script

    ```bash
    npm run watch:dev
    # or
    npm run build:dev
    ```

    Change "dev" to "prod" if you want to run the API against flattr.com instead of flattr.test.

1. Load `/dist` as an unpacked extension in your browser.
1. Success!
